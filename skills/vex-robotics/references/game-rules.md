| Program | Age Group | Robot Size | Key Feature |
|---------|-----------|------------|-------------|

## VEX V5 Robotics Competition (VRC) — Core Format

### Season Structure
- **New game revealed**: Late April each year (VEX Worlds closing ceremony)
- **Season runs**: September → April (State/Regional → World Championship)
- **Game changes every year** — always verify the current season's game manual

### Match Format
Each match (2 minutes total):
1. **Autonomous Period** (15 seconds): Pre-programmed only, no driver input
2. **Driver-Controlled Period** (1 minute 45 seconds): Drivers control robots via joysticks
3. **Alliances**: Two robots per alliance (Red vs. Blue), 4 teams total per match

### Scoring Concepts (Vary by Game — Examples)
- Scoring elements: rings, cubes, goals, mobile goals, triball, stakes, etc.
- **Autonomous Win Point (AWP)**: Bonus for winning autonomous period
- **Elevation/Hanging**: End-of-match bonus for climbing structures
- Descoring: some games allow removing opponent's scored objects

### Tournament Structure
- **Qualification Rounds**: Swiss-style seeding, each team plays ~5–8 matches
- **Alliance Selection**: Top 8 teams pick partners (snake draft)
- **Elimination Rounds**: Best-of-3 bracket (Quarterfinals → Semifinals → Finals)

### Awards (Common at Most Tournaments)
- **Tournament Champions**: Win the elimination bracket
- **Excellence Award**: Best overall team (highest priority award for advancement)
- **Design Award**: Outstanding engineering design notebook
- **Skills Champion**: Best combined Autonomous + Driver skills score
- **Judges Award**: Special recognition at judges' discretion

### Skills Challenge Format
- **Autonomous Skills**: 60-second run with pre-programmed robot, no driver
- **Driver Skills**: 60-second run with single driver, no alliance partner
- **Score = best Autonomous + best Driver scores combined**
- Skills rankings used for World Championship qualification

## VEX IQ Competition

### Format
- **Teamwork Challenge**: Two teams cooperate (no opposing alliance)
- **Robot Skills**: Individual team runs (Autonomous + Driving)
- No elimination brackets — ranking based on highest combined scores

### Key Difference from VRC
Cooperative rather than competitive alliance format; emphasizes collaboration.

## Engineering Design Notebook

A critical component judged for awards like Excellence and Design:

### Required Sections
1. **Team Profile**: Member roles, bios, meeting schedule
2. **Game Analysis**: Breakdown of rules, scoring elements, strategy brainstorm
3. **Design Process**: Brainstorm → Prototype → Build → Test → Iterate loop
4. **Build Documentation**: Photos, CAD drawings, measurements, materials list
5. **Programming Log**: Algorithm explanations, pseudocode, test results
6. **Competition Reflections**: Post-match analysis, lessons learned

### Judging Criteria
- Follows Engineering Design Process (not just a diary)
- Regular entries with dates (shows ongoing work, not last-minute fill)
- Evidence of iteration and testing
- Student-authored (judges can ask follow-up questions)

## Robot Rules & Constraints (VRC — General)

> Always verify exact dimensions and weight limits in the current season's game manual.

### Typical Size Constraints
- **Starting size**: 18" × 18" × 18" (457mm × 457mm × 457mm)
- **Expansion**: Some games allow robots to expand beyond starting size after match begins

### Allowed Components
- VEX V5 electronics only (motors, sensors, brain, controller)
- VEX-legal structural materials (metal, plastic, rubber bands, pneumatics)
- Custom non-functional decorations allowed (must not affect performance)

### Disqualification Triggers
- Robots that are unsafe or damage field elements
- Intentional entanglement or pinning (pinning > 5 seconds = violation warning)
- Match-affecting rules violations result in DQ for the match

## Common Field Positions

### Scouting Terminology
- **OPR (Offensive Power Rating)**: Average contribution of a single robot to the alliance score per match (derived stat; higher = better)
- **DP (Defensive Pressure)**: Estimated negative impact on opponent scoring (useful for alliance selection against top teams)
- **Autonomous Routine**: Team's pre-programmed 15-second sequence
- **High Goal / Low Goal**: Different scoring targets at different point values
- **Neutral Zone**: Central area often contested by both alliances

### Robot Role Archetypes (Vary by Season)
- **Scorer**: Focuses on placing/scoring game objects
- **Defender**: Disrupts opponent's scoring
- **Hybrid**: Balanced score and defense
- **Autonomous Specialist**: Optimized for 15-second autonomous bonus

## World Championship Qualification Pathways

1. **Skills Rankings**: Top teams nationally via skills leaderboard (robotevents.com)
2. **Excellence Award**: Winners at qualifying tournaments earn a bid
3. **Tournament Champions**: Regional/State champions often receive bids
4. **Host Region**: Guaranteed spots for local teams

Check **robotevents.com** for current season qualification criteria.
